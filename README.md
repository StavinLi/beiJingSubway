## beiJingSubway
北京地铁图

### 项目运行
- 环境依赖
    1. node v6.10.2

- 部署步骤
    1. npm i  //安装
    2. npm run start //运行
    3. npm run build //构建编译

### 目录结构描述
```
├── Readme.md                   //help
├── dest                        //发布包
│   ├── css
│   ├── js                
│   └── *.html        
├── libs                        //第三方文件
├── node_modules                  
├── rev                         //静态版本json
├── src                         //开发包
└── gulpfile.js
```
